第N个丑数

发表于 2017-03-20 | 分类于 算法 | 阅读数

题目描述

  把只包含因子2、3和5的数称作丑数（Ugly Number）。例如6、8都是丑数，但14不是，因为它包含因子7。习惯上
我们把1当做是第一个丑数。求按从小到大的顺序的第N个丑数。

思路

  暴力求解思路。首先我们要搞清楚怎么求丑数。ugly[]表示丑数数组。

  1. ugly[0]=1;
  2. ugly[1]=1*2;
  3. ugly[3]=1*3;
  4. ugly[4]=122;
  5. ugly[5]=1*5;
  6. ugly[6]=123;
  7. …
      是不是可以找出规律来，我们需要记录 丑数中因子 2,3,5 出现的次数。具体用语言不好描述，可以看出规律。
